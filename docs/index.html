Docker Quick Start (Development)

For a containerized development environment, Docker provides a simple and isolated setup. Note that this configuration uses an ephemeral database.

1. Build the Image

This command packages the application and its dependencies into a Docker image.

docker build -t promptopia-demo-sqlite:latest .

2. Run the Container

This command starts the container, mapping port 8080 and injecting the required session secret.

docker run -p 8080:3000 -e SESSION_SECRET="..." promptopia-demo-sqlite:latest
